# RP-init
SCP: CS file for plugins on lua

Этот файл необходимо установить в папку `Plugins`, чтобы плагины для РП на lua, приведённые здесь, работали.

**Примечание:** файл переименовывать нельзя! Он должен называться `init.lua`.

Этот файл необходим для работы плагинов из следующих репозиториев:
* AdditionalAdminPanel;
* UpdatedRooms;
* RoundTimer;
* NewRoundManager;
* NewSupportManager;
* Tips;
* AutoSpawn;
* WaitingPVP;
* SCP173CageBoxSpawner;
* SCP-1162;
* TutorialClass;
* SerpentsHand;
* NewChaosInsurgency;
* SCP420J+;
* Cup+;
* SCP173Cage.
